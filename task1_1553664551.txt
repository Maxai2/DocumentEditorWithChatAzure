Сетевая записная книжка, предоставляющая следующие возможности:
- регистрация, логин
- создание новых документов
- редактирование документов
- удаление документов
- создание read-only ссылки на свои документы, чтобы поделиться ими
- предусмотреть возможность вставлять изображения в текст


Azure Web App - для хостинга серверного приложения на ASP.NET Core Web API
Azure Web App - для хостинга клиентского приложения на Angular
Azure SQL Database - для хранения информации о созданных пользователями документах
Azure Blob Storage - для хранения самих документов и изображений
